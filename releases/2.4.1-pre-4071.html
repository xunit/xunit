Release Notes: October 6, 2018

xUnit.net 2.4.1 Pre-Release (build 4071)

Core framework

  • BUG: Fixed an issue caused in 2.4 by introducing new netstandard2.0 libraries. Developers who were using desktop versions older than 4.7.2 could encounter assembly loading issues. We rolled this change back, which should resolve the issue for those users. For more information on this issue, see this Github issue.
  • BUG: Fixed an issue where generic test methods weren't correctly resolved when passing a Func<T>.

Console runner

  • BUG: Fixed an issue where -nocolor was not honored (when trying to get any reports other than XML v2 from the .NET Core 1.x runner).
